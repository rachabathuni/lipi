sailu.com's Lipi తెలుగులో రాసే సదుపాయం కల్పిస్తుంది. కింద కనిపిస్తున్న డబ్బాలో తెలుగులో రాస్తున్నట్టు రాయండి, మీకు తెలుగు అక్షరాలు కనిపిస్తాయి. ఎలా రాయాలో కింద కొన్ని ఉదాహరణలు చూడండి. ఎంత తేలికో మీకే అర్ధం అవుతుంది. ఈ రాసే పద్దతిని RTS పద్దతి అంటారు‌. పక్కన ఉన్న RTS Helpలో పూర్తి వివరాలు చూడండి.

మీరు రాసిన తెలుగు వాక్యాలని మార్చచ్చు, తప్పులు సరిదిద్దుకోవచ్చు. మీకు నచ్చినట్టుగా వచ్చాక, మొత్తం copy చేసి మీకు కావలిసిన programలో paste చేసుకోవచ్చు.

ఈ లిపిని వాడటానికి అధునాతనమైన browser ఉండాలి. సరిగా పని చెయ్యకపోతే మీ browser update అవసరం ఉందేమో చూడగలరు.

మీ సలహాలు, సందేహాలు, విన్నపాలు, చీవాట్లు ఇచ్చటికి పంపగలరు: lipi@sailu.com. ఈ editorని మరింత మందికి ఉపయోగపడేలా తీర్చిదిద్దేందుకు సహాయపడగలరు.

Intro in English

Sailu.com's Lipi is an online Telugu Unicode text editor. It allows you to write and edit Telugu text right in your browser.

You can type Telugu using English letters. RTS is a popular transliteration scheme. You can find complete list of letter translations in "RTS Help" section.

This editor requires a modern browser to work since it uses some of the advanced features of browser. Please make sure you have the latest version of the browser.

Please send your feedback, bug reports and feature requests to: lipi@sailu.com. Please help me make this editor better.

Auto-Save

Auto-save వాడితే మీరు రాసినదంతా ప్రతి కొద్ది క్షణాలకి మీ browserలో save చెయ్యబడుతుంది. మీరు browser మూసివేసినా, మళ్ళీ తెరిచినప్పుడు మీరు రాసినది మళ్ళీ కనిపిస్తుంది. మీరు రాసింది ఈ computerలో మాత్రమే ఉంటుంది అని గమనించగలరు.
