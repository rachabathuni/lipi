<h1>Lipi Inline Telugu Editor</h1>
Lipi allows you to type in Telugu using the standard RTS transliteration.

<h2>Example</h2>
To get: <br/>
లిపి వాడి మీరు తెలుగులో వ్రాయచ్చు
<br/>
Type:
lipi vADi mIru telugulO vrAyachchu
